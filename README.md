# gsdown

## Google Site gadget for Markdown Text

### Using

* Click [Insert]
* Select [More...]
* Click [add by url]
* Copy and paste the url to https://raw.githubusercontent.com/fabianofranca/gsdown/v1.0-beta/gsdown.xml
